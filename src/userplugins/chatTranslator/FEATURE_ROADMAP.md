# ChatTranslator Feature Roadmap

이 문서는 2026-04-11 기준 `chatTranslator` 플러그인에 추가하기로 한 UX/기능 작업을 정리한 문서입니다.

## 구현 목록

### 1. 번역 결과 접기/펼치기
- 기본 접힘 설정 추가
- 메시지별 펼침/접힘 상태 기억
- 펼침/접힘 버튼 제공

### 2. 원문/번역문 표시 방식 선택
- 번역문만 표시
- 번역문 + 원문 표시
- 원문/번역문 토글 표시

### 3. 길이 제한 / 긴 메시지 자동 스킵
- 자동 번역 최대 글자 수 설정
- 자동 번역 최대 줄 수 설정
- 코드 블록 포함 메시지 자동 스킵
- 수동 번역은 계속 허용

### 4. 채널별 언어 설정
- 채널별 입력 언어 오버라이드
- 채널별 출력 언어 오버라이드
- 글로벌 언어 설정으로 되돌리기

### 5. 번역 실패 사유 간단 표시
- 메시지 단위 skip reason 표시
- 메시지 단위 failure reason 표시
- 수동 재시도 / 강제 번역 진입점 유지

### 6. 번역 캐시 관리 UI 보강
- 현재 캐시 개수 표시
- 현재 설정과 일치하는 캐시 개수 표시
- 마지막 캐시 시각 표시
- 현재 캐시 시그니처 상태 표시
- 오래된 캐시 자동 만료 설정
- 캐시 통계 수동 새로고침, 현재 언어 캐시만 삭제, 만료 캐시만 정리, 전체 캐시 삭제 액션 제공

### 7. 채널 컨텍스트 메뉴 빠른 액션
- 이 채널만 한국어로 번역
- 이 채널 번역 켜기/끄기
- 이 채널 캐시 지우기

### 추가 UX 정리
- 무시 목록은 ID만 보여주지 않고, Discord에 로드된 서버/채널/유저 이름을 함께 표시한다.
- 이름을 알 수 없는 항목은 Unknown server/channel/user로 표시하고 ID는 그대로 남겨 직접 확인할 수 있게 한다.
- 무시 목록은 서버/채널/현재 서버 유저/최근 DM 유저를 검색해서 추가할 수 있게 하고, 현재 채널/서버 빠른 추가 버튼을 제공한다.

## 구현 메모

- 링크, 커스텀 이모티콘, 멘션, 타임스탬프, 코드 블록은 번역 전에 보호하고 이후 복원한다.
- 긴 메시지나 코드 블록은 자동 번역만 건너뛰고, 수동 번역은 가능하게 유지한다.
- 채널별 언어 설정은 글로벌 기본값 위에 오버라이드 방식으로 동작한다.
- 캐시 만료일 설정은 0이면 비활성화되고, 기본값은 14일이다.
- 캐시 만료는 설정 변경뿐 아니라 다음 만료 시각에 맞춰 자동 정리 타이머로도 처리한다.
